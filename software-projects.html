Eklavya: Software Development Projects

  1. cpigjs/fairDiv

    (April 2024 – Dec 2024) source code: github:sharmaeklavya2/cpigjs, paper: arXiv:2502.02815.

    Web interface to explore relations between fairness notions for fair division of indivisible items.

  2. code2dtree

    (Sept 2023 – March 2024) paper: arXiv:2504.04738

    Converts Python functions to decision trees. Was created to help find hard examples for approximation algorithms.

  3. TeX tools

    1. TeX-flatten: Convert a multi-file TeX project into a single TeX file.
    2. TeX-filter: Remove unneeded info from TeX's stdout.
    3. tex2text: Convert TeX code to plain text.
    4. TeX-colorscheme: Seamlessly switch between color schemes (light, dark, sepia) in LaTeX documents.
    5. download-latex-files.py: Automates retrieval of external class files (e.g. LIPIcs).

  4. Trin

    (March 2023)

    Transliterate among Indian languages. Available as a web app, a JavaScript library, and a browser extension.

  5. Packing Game

    (May 2020)

    A 2D geometric bin-packing game that runs in the browser.

  6. ConcepDAG

    (Nov 2018 – Feb 2019)
    • ConcepDAG is a tool to visualize dependencies between concepts in the form of a static website.
    • I used ConcepDAG to create TheoremDep.

  7. GSoC at Zulip

    (May 2016 – Aug 2016) Mentor: Tim Abbott.
    • Added mypy type annotations to Zulip's python code (~50K lines).
    • Made Zulip's code Python 3 compatible.
    • My work was mentioned on the Google Open Source Blog.
    • See my GSoC output summary.

  8. Internet Whirlwind

    (Jan 2016 – March 2016)

    Web application for an online quiz organized by BITS-ACM.

  9. Checkmate

    (Aug 2015 – Sept 2015)

    Web application for an online quiz organized by BITS-ACM. It features webglearth.
