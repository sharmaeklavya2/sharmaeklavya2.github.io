Eklavya: Software Development Projects

  1. TeX tools

    1. TeX-flatten: Convert a multi-file TeX project into a single TeX file.
    2. TeX-filter: Remove unneeded info from TeX's stdout.
    3. tex2text: Convert TeX code to plain text.
    4. TeX-colorscheme: Seamlessly switch between color schemes (light, dark, sepia) in LaTeX documents.
    5. download-latex-files.py: Automates retrieval of external class files (e.g. LIPIcs).
